Summary: Latitude vs. X



The purpose of this project was to analyse how the weather changes as getting closer to the equator (0 degree lattitude). To accomplish this analysis, the data from the Open Weather Map API was pulled via API and formed a dataset of over 500 cities within the radius of -90 to 90 degree lattitude .
Pandas and Matplotlib were used to plot various aspects of the weather vs. latitude. Factors to analyse included: temperature, cloudiness, wind speed, and humidity (denotes by X). This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
